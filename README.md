# cmb-scale-invariants

**Paper title:** Robust Scale-Dependent Structure in the Cosmic Microwave Background: An Information-Theoretic Diagnostic

This repository contains code, documentation, and artifacts for an operational, information-theoretic diagnostic applied to Planck CMB products (e.g., SMICA, NILC), including robustness checks across masks, hemispheres, and half-mission splits.

## Repository layout
- `paper/` manuscript sources and figures
- `src/` core library code
- `scripts/` command-line entry points
- `data/` raw and processed data (not committed)
- `reports/` generated tables/figures (not committed, except placeholders)
- `docs/` reproducibility and data source notes
